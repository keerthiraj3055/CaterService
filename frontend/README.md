# FoodServe Frontend

This is the frontend for the FoodServe catering service MERN application.

## Features
- User authentication
- Menu browsing and booking
- Order management
- Payment integration
- Reviews and testimonials
- Dashboards for admin, user, corporate, and employee roles
- Analytics and reporting

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Folder Structure
- `public/` - Static files
- `src/api/` - API setup
- `src/assets/` - Images, icons, styles
- `src/components/` - Reusable React components
- `src/pages/` - Page components
- `src/dashboards/` - Dashboard components
- `src/context/` - React context providers
- `src/hooks/` - Custom hooks
- `src/utils/` - Utility functions
